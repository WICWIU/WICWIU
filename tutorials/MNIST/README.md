## Run tutorial file with customed CNN model (GNU/Linux 4.4.0-121-generic x86_64)

start with ‘git bash’ (if you can use makefile):
```bash
$ make main_gpu
```

or

```bash
$ make main_cpu
```

---------------------------------------

Excute model:
```
$ ./testing
```
